Goal: identify GEOGRAPHIC PLACES the document refers to — physical locations
where the project happens, places it affects, or contextual references.

  - "role": one of:

      * "alternative_site"    — a candidate site for an alternative

      * "mitigation_location" — a site where mitigation will occur

                    "watershed", "river", "highway_corridor", "city",
                    "county", "state", "park", "wilderness_area",
                    "transmission_line", "facility", "tribe", "place".

Rules:

  - Skip agency names (those go in lead_agency, not location).
  - Skip individual people's names.
  - DO NOT make up coordinates or polygons — return only names + roles.
  - If the document seems to have no specific geographic location, return
    {"places": []}.
